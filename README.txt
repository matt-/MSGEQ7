This is an Arduino library for Bliptronics MSGEQ7 Seven Band Spectrum Analyzer Shield
This is NOT provided by Bliptronics. Use at your own risk. 

Pick some up at http://www.bliptronics.com/item.aspx?ItemID=116

To download. click the DOWNLOADS button in the top right corner, rename the uncompressed folder MSGEQ7. Check that the MSGEQ7 folder contains MSGEQ7.cpp and MSGEQ7.h

Place the MSGEQ7 library folder your <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.
