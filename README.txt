This is an Arduino library for Bliptronics MSGEQ7 Seven Band Spectrum Analyzer Shield 

To Install: 
Place the MSGEQ7 library folder your <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.


Pick one up at http://www.bliptronics.com/item.aspx?ItemID=116
